# IndiEthnicBazar - Full-Stack E-commerce Website with Admin Panel


IndiEthnicBazar is a comprehensive full-stack e-commerce website built using Apache, PHP, MySQL, and Bootstrap. The platform includes an admin panel for efficient management of products, orders, and user accounts.

## Features

- **User Authentication:** Secure user authentication ensures only authorized users can access and shop on the platform.

- **Product Details:** Detailed product pages provide images, descriptions, and pricing for each item.

- **Add to Cart and Checkout:** Users can add products to their cart and proceed to checkout securely.

- **User Profiles:** Each user has a profile where they can view order history and manage account details.

- **Admin Panel:** The admin panel allows administrators to manage products, orders, and user accounts efficiently.

- **Authorization System:** Role-based authorization ensures only authorized admin users can access and manage the admin panel.

- **Responsive Design:** The website features a responsive design, offering optimal browsing and shopping experiences on various devices.

## Technologies Used

- **Frontend:** HTML, CSS, Bootstrap
- **Backend:** Apache, PHP
- **Database:** MySQL

## Installation and Setup

1. Clone this repository to your local server or hosting environment.
2. Set up your Apache server to point to the project directory.
3. Import the provided MySQL database schema and sample data.
4. Configure database connection settings in `config.php`.
5. Access the admin panel by visiting `/admin` and logging in with admin credentials.
6. Start browsing and shopping on the platform!
